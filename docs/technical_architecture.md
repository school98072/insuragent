# 技术架构设计

## 🏗️ 整体架构

### 系统层次架构
```
┌─────────────────────────────────────────────────┐
│                客户端层 (Frontend)               │
│  ├── Web应用 (React + TypeScript)               │
│  ├── 移动端 (React Native)                      │
│  └── 管理后台 (Next.js)                         │
└─────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│                API网关层 (Gateway)               │
│  ├── 请求路由和负载均衡                         │
│  ├── 认证授权 (JWT/OAuth2.0)                    │
│  ├── 流量控制和熔断                             │
│  └── 日志记录和监控                             │
└─────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│              业务服务层 (Services)               │
│  ├── 报案服务                                   │
│  ├── 审核服务                                   │
│  ├── 定损服务                                   │
│  ├── 知识库服务                                 │
│  └── 用户管理服务                               │
└─────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│                 AI核心层 (AI Core)               │
│  ├── 多Agent协调引擎                            │
│  ├── LLM对话服务                                │
│  ├── OCR识别服务                                │
│  ├── 多模态分析服务                             │
│  ├── RAG检索服务                                │
│  └── 模型微调服务                               │
└─────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│               数据存储层 (Data Layer)            │
│  ├── 关系数据库 (PostgreSQL)                     │
│  ├── 文档数据库 (MongoDB)                        │
│  ├── 向量数据库 (Pinecone/Weaviate)              │
│  ├── 缓存系统 (Redis)                            │
│  └── 文件存储 (MinIO/S3)                         │
└─────────────────────────────────────────────────┘
```

## 🎯 AI系统架构

### 多Agent协同系统
```
                              ┌─────────────────┐
                              │   用户请求入口   │
                              └─────────────────┘
                                       ↓
                              ┌─────────────────┐
                              │   协调器Agent   │
                              └─────────────────┘
                               ┌──────────────────┴─────────────────┐
                               ↓                                    ↓
                    ┌─────────────────────┐             ┌─────────────────────┐
                    │    定损Agent        │             │    条款Agent        │
                    │  • 图像分析         │             │  • 条款解读         │
                    │  • 损失评估         │             │• 责任匹配         │
                    │  • 维修估算         │             │  • 费率计算         │
                    └─────────────────────┘             └─────────────────────┘
                               │                                    │
                               └─────────────────┬──────────────────┘
                                                 ↓
                                      ┌─────────────────────┐
                                      │    决策Agent        │
                                      │  • 综合评估         │
                                      │  • 风险识别         │
                                      │  • 最终决策         │
                                      │  • 人工转介判断     │
                                      └─────────────────────┘
                                                 ↓
                                      ┌─────────────────────┐
                                      │    输出结果         │
                                      └─────────────────────┘
```
